## 2025-11-18 - Navigation in Long Markdown Documents
**Learning:** In documentation-heavy repositories without a generated site (just raw markdown), navigation becomes a significant friction point. Users reading long files (like frameworks or roadmaps) lose context when scrolling.
**Action:** Consistently add `[Back to Top](#table-of-contents)` links before section separators (`---`). This simple pattern significantly improves navigability without requiring complex tools or a documentation site generator.
