## 2025-05-20 - Navigation in Long Documents
**Learning:** In long Markdown documentation files without a generated site (raw GitHub rendering), users struggle to navigate between sections. Standard navigation aids (ToC + Back to Top links) significantly improve readability and accessibility.
**Action:** When auditing documentation repos, check for ToC and Back to Top links in files > 300 lines. Use scripts to insert them consistently to avoid manual errors and ensure consistent anchor generation.
