## 2025-05-22 - [Enhancing Navigation in Long Markdown Files]
**Learning:** Users often get lost in long markdown documents without navigation aids like 'Back to Top' links, which are especially critical in environments without a generated sidebar (like raw GitHub file view).
**Action:** Automatically insert anchor tags at the top and 'Back to Top' links at the end of every major section in long documentation files to improve usability.
